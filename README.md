# youtube-downloader
Download all videos from YT channel and cutting audio

How it works:
- ./download_channel_audio.sh **url** **path** — download all video from channel and convert to mp3
- ./download_channel_audio.sh -v **url** **path** — dowbload all video from channel
For playlist:
- ./download_playlist_audio.sh **url** **path** — sownload playlist and convert to mp3
- ./download_playlist_audio.sh -v **url** **path** — download playlist
